Answer the following questions

  1. What did you do the apply the CSS style onto the given html contents?

    Ans ทำการใส่ link stylesheet ที่เป็นการ import external file เข้าไปในแต่ละ webpage โดยแยกเป็นแต่ละ style

  2. What did you do to make Thai text shown correctly?

    Ans เปลี่ยน <meta http-equiv="content-type" content="text/html; charset=windows-1252">
        เป็น <meta http-equiv="content-type" content="text/html; charset=UTF-8">

  3. What did you do to make the question and the corresponding answer appear as shown in the example?

    Ans ใช้ html Element <p> <u> และ Special Charactor ในการประกอบเนื้อหาเข้าด้วยกัน โดยทั้งหมดนี้จะต้องอยู่ภายใต้
        html Element <ol> หากต้องการเพิ่มข้อคำถามและคำตอบ ให้เพิ่ม html Element <li>

  4. How was the stylesheet option selected correctly for each page?

    Ans Link stylesheet ให้ถูกต้อง โดยดูจากชื่อ และ content css ภายในไฟล์นั้นว่า html Element แต่ละตัวมีการใส่ไว้อย่างไร

  5. What did you do to make the images laid out in one row like in the example?

    Ans ใช้style="display: flex; flex-direction: row;โดยมีการจัดให้อยู่กึ่งกลางโดยใช้ align เป็น center
